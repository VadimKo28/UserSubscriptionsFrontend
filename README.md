## User Subscriptions Frontend

Небольшое React-приложение для создания нового пользователя.

### Установка

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

После запуска приложение будет доступно по адресу `http://localhost:3010`.

### Функциональность

- **Форма создания пользователя**:
  - Поля: **Имя**, **Почта**, **Пароль**
  - Отправляет `POST` запрос на `http://localhost:8081/users/sign-up` с телом:

    ```json
    {
      "name": "Имя",
      "email": "Почта",
      "password": "Пароль"
    }
    ```

- При успешном ответе отображается сообщение об успехе.
- При ошибке — сообщение с текстом ошибки.
